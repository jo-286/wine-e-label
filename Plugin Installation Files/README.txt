PLUGIN INSTALLATION FILES

This folder contains the installable WordPress ZIP files for this project.

WHICH PLUGIN GOES WHERE?

1. wine-e-label.zip

Install this plugin on the main WordPress or WooCommerce website where you manage your wine products.

Use it there for:
- product editor workflow
- import of ZIP, JSON, or HTML source data
- manual correction of label data
- local e-label generation
- QR code generation
- optional publishing to a remote receiver

2. wine-e-label-receiver.zip

Install this plugin only on a separate WordPress site that is intended to act as the receiver instance.

Use it there for:
- receiving label data from the main plugin via REST API
- hosting clean standalone e-label pages
- separating the public label output from the main shop site

3. Testdata for Zip Import.zip

This ZIP file contains test data for checking the import workflow and for testing whether your setup is working as expected.

Do not install this file as a WordPress plugin.

Use it only as sample input data inside the main Wine E-Label plugin.

TYPICAL SETUP

- Main shop site: install wine-e-label.zip
- Separate receiver site: install wine-e-label-receiver.zip

If you only want to run labels locally on the same website, you usually only need the main plugin.


PLUGIN-INSTALLATIONSDATEIEN

Dieser Ordner enthält die installierbaren WordPress-ZIP-Dateien für dieses Projekt.

WELCHES PLUGIN WIRD WO INSTALLIERT?

1. wine-e-label.zip

Dieses Plugin wird auf der Haupt-WordPress- oder WooCommerce-Seite installiert, auf der die Weinprodukte verwaltet werden.

Dort wird es verwendet für:
- den Workflow im Produkteditor
- den Import von ZIP-, JSON- oder HTML-Quelldaten
- die manuelle Korrektur der E-Label-Daten
- die lokale E-Label-Erzeugung
- die QR-Code-Erzeugung
- die optionale Veröffentlichung an einen separaten Receiver

2. wine-e-label-receiver.zip

Dieses Plugin wird nur auf einer separaten WordPress-Seite installiert, die als Receiver-Instanz dienen soll.

Dort wird es verwendet für:
- das Empfangen der E-Label-Daten vom Hauptplugin per REST API
- das Hosting eigenständiger reduzierter E-Label-Seiten
- die Trennung der öffentlichen E-Label-Ausgabe von der eigentlichen Shop-Seite

3. Testdata for Zip Import.zip

Diese ZIP-Datei enthält Testdaten, um den Import-Workflow zu prüfen und um zu testen, ob dein Setup wie erwartet funktioniert.

Diese Datei wird nicht als WordPress-Plugin installiert.

Sie wird nur als Beispiel- und Testdatei innerhalb des Hauptplugins Wine E-Label verwendet.

TYPISCHE EINRICHTUNG

- Haupt-Shop-Seite: wine-e-label.zip installieren
- Separate Receiver-Seite: wine-e-label-receiver.zip installieren

Wenn die E-Labels nur lokal auf derselben Website betrieben werden sollen, reicht in der Regel nur das Hauptplugin.
